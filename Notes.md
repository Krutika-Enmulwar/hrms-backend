What is this project?
A basic HRMS (Human Resource Management System) to manage employees in a company. 

Who are the users?
1. Admin 
Admin controls the system. 
Admin can: 
   - add new employees.
   - view all employees.
   - assign departments. 
   - approve or reject leave requests.
   - view attendance of employees.  

2. Employee 
Employee is a normal user. 
Employee can: 
   - log in/log out. 
   - view their own profile. 
   - apply for leave. 
   - punch in and punch out for attendance. 
   - view their attendance and leave status. 

Main features: 
1. Login and Authentication 
   - users log in using email and password. 
   - system knows if user is admin or employee. 
   - only admins can access admin features. 

2. Employee management: 
   - admin can add employees. 
   - each employee has name, email, role, department. 
   - employees can view their own details, edit a few. 

3. Department management:
   - admin can create departments. 
   - employees belong to a department. 

4. Leave management: 
   - employees can apply for leave. 
   - admin can approve or reject. 
   - leave has status - pending/approved/rejected. 

5. Attendance: 
   - employee can punch in/out. 
   - system stores date and time. 
   - admin can view attendance records.
